Pending and limits

  • Forecast fallback map is marked pending because the reliable view is the real Leaflet map with sampled live points.
  • Warnings are marked pending until DHMZ CAP is wired as a live browser or backend source for the selected location.
  • RainViewer radar is a near-now map overlay only; future rain timing still comes from forecast providers and verification history.
  • The Mediterranean grid is visual context for wind/rain fields and remains weight 0 until bias checks against observations are available.
  • Open-Meteo Marine current and tide-like fields are planning context only near coasts; official nautical sources are still required.
  • Official Croatian ENC/HHI chart data is not bundled and must be licensed before navigation-grade claims.
